## Instructions
### Arduino
1. Launch Arduino IDE
2. Set the Serial Port in the IDE: Tools / Port / (Arduino)
3. Select File / Examples / Firmata / Standard Firmata
4. Select Sketch / Upload

### Server
1. `npm install`
2. `node index.js`
